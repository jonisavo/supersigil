## Overview

Implementation tasks for ecosystem plugins, ordered by dependency and written
for TDD execution. Each implementation slice starts with failing tests, then
the implementation, then integration wiring. Keep the feature work bounded to
the new evidence layer: `DocumentGraph` remains document-only while verification
gains `ArtifactGraph` enrichment.

Use `cargo nextest run` as the default test command while implementing. Run
`cargo fmt --all`, `cargo clippy --workspace --all-targets --all-features`, and
`cargo nextest run` before closing the implementation phase.
