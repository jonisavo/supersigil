# supersigil-cli

CLI tool for [supersigil](https://github.com/jonisavo/supersigil):
spec-driven development with AI agents.

This crate provides the `supersigil` binary and its command implementations.

## Installation

```sh
cargo install supersigil-cli
```

## Usage

```sh
# Initialize a new supersigil project
supersigil init

# Verify specs against code
supersigil verify

# See all commands
supersigil --help
```

For full documentation, visit [supersigil.org](https://supersigil.org).
