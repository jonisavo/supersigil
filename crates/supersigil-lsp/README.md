# supersigil-lsp

Language Server Protocol implementation for
[supersigil](https://github.com/jonisavo/supersigil) spec documents.

Provides IDE features such as diagnostics, completions, code actions, go to
definition, hover, and rename for supersigil spec documents. Used by the
VS Code and IntelliJ extensions.

## Installation

```sh
cargo install supersigil-lsp
```

Editor extensions typically bundle or install the binary automatically.
